# TASUED SmartAttend Component Inventory

## Shared UI Components

- `Button`: Primary/secondary/ghost actions for forms, nav, and CTA areas.
- `DashboardShell`: Reusable role-based shell with sidebar, mobile menu, title bar, and content panel.
- `KpiCard` (planned): Compact metric card for dashboard overviews.
- `DataTable` (planned): Searchable, filterable table wrapper for students, courses, sessions, and attendance lists.
- `StatusBadge` (planned): Uniform status rendering for active/closed sessions and attendance outcomes.

## QR Workflow Components

- `QRCodePanel` (planned): Lecturer-facing card with generated QR, session metadata, and TTL indicator.
- `ScannerViewport` (planned): Student-facing camera viewport wrapper around `html5-qrcode`.
- `ScanFeedback` (planned): Success/error feedback card with clear next actions.

## Reporting Components

- `ChartCard` (planned): Wrapper for `Recharts` visualizations with title and legend slots.
- `DateRangeFilter` (planned): Reusable date/course filter bar for reports and history.
- `ExportActions` (planned): CSV export and print actions with access checks.

## Marketing/Home Components

- `BrandHeader`: Logo and TASUED SmartAttend identity block.
- `WorkflowSteps`: Four-step process strip for homepage explanation.
- `TeamMemberCard`: Student profile card with image, name, and role.
- `SupervisorCard`: Professional section for project supervisor details.
